# handoff

Use when:
- You are delivering the final implementation summary.
- You need to present validation and exact commands.

Workflow:
1. Summarize the work done.
2. List files changed and why.
3. Provide validation commands and outputs.
4. Note any risks or follow-up items.

Expected output:
- A clear handoff message.
- Exact evidence commands.
- Recommended next steps.
