# triage

Use when:
- Incoming requests need classification or missing context.
- You need to decide whether a task is ready for agent work.

Workflow:
1. Review the request or issue.
2. Identify missing information.
3. Assign readiness and labels.
4. Recommend the next skill or action.

Expected output:
- A triage decision.
- Required follow-up information.
- Suggested labels and readiness state.
