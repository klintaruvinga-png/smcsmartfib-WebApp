# Frontend load fix evidence - 2026-06-23

## Failure observed

Production HTML loaded from:

`https://smcsuperfibwebapp.klintaruvinga.workers.dev/`

but static assets under `/assets/*` returned 404, including:

- `/assets/styles-B_y_4r30.css`
- `/assets/index-ByjDYHRf.js`

This meant the Worker was serving HTML that referenced Vite/TanStack client assets, but Wrangler was not deploying `dist/client` as static assets.

## Fix

Updated `wrangler.jsonc` to bind the generated client asset directory:

```json
"assets": {
  "directory": "./dist/client"
}
```
