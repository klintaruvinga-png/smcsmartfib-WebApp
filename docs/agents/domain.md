# SMC SuperFIB Domain Context

This glossary defines key terms and domain concepts for the repo.

Visual authority
- The Pine indicator is the visual authority during MT5 migration.
- Do not treat Pine behavior as legacy while it remains the authoritative reference.

Authority Fib
- The fib levels generated by Pine that are authoritative for parity validation.
- Backend/EA fibs must match these before parity gates can pass.

SuperFib
- The combined SMC-specific fib system used by the dashboard and migration.
- Includes AF/SF anchors, HTF/LTF rules, and chop-block logic.

Parity gate
- The project checkpoint where Pine and MT5/backend fib generation must match.
- Failure at the parity gate blocks migration progress.

Market stream
- The EA-to-backend data feed route for live market updates.
- Includes symbols, price ticks, and state events.

Heartbeat
- EA health pings sent to the backend.
- Used to detect stale feeds and confirm connectivity.

Display signal
- Dashboard-visible signal state derived from backend/display data.
- Not the same as execution state.

Grace window
- A short tolerance period for stale data before the system alerts.
- Used to avoid false failure reports during transient connection issues.

Plan card
- Dashboard UI component showing next-best blueprint, RR display, and execution details.
- Must respect account currency, user local currency, and symbol-specific lot rules.

Account currency
- The trading account’s settled currency.
- Different from the user’s local display currency.

User local currency
- The currency shown to the user for PnL, RR, and risk display.
- Conversion logic must be explicit and correct.

Symbol-specific lot minimum
- Minimum executable lot size for a given symbol.
- Do not assume forex lot rules are valid for gold, crypto, or indices without symbol-specific rules.

Avoid language
- Do not call Pine behavior “legacy” while it remains visual authority.
- Do not call backend fibs “correct” unless parity has been proven.
- Do not treat forex lot minimums as valid for gold, crypto, or indices without symbol-specific rules.
