# grill-with-docs

Use when:
- The task requires a deep review of repo documentation, reports, or architecture notes.
- You need to compare current behavior with documented intent.

Workflow:
1. Read `AGENTS.md`, `docs/agents/*`, and any relevant reports.
2. Identify the feature or process in question.
3. Compare implementation, docs, and existing guidance.
4. Produce a focused recommendation or plan.

Expected output:
- Evidence-backed discrepancies.
- Clear reference to docs and repo artifacts.
- A suggested implementation path.
