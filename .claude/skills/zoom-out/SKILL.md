# zoom-out

Use when:
- High-level planning or scope alignment is needed.
- You need to map dependencies across Pine, MT5 EA, backend, and dashboard.

Workflow:
1. Review current issues and dependencies.
2. Map the problem across layers.
3. Propose a prioritized approach.
4. Document tradeoffs.

Expected output:
- A project-level summary.
- Recommended priorities.
- A concise roadmap.
